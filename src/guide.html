Node.js 설치

10분

Node.js 설치

Node.jsJavaScript를 브라우저 밖에서도 실행할 수 있게 해주는 프로그램는 Claude Code가 작동하는 데 필요한 프로그램입니다.

먼저 적절한 버전의 Node.js가 설치되어 있는지 확인합니다:

node --version

버전이 18이상이면 확인을 누르고 다음 단계로 넘어가세요. node가 설치되어 있지 않거나, 버전이 18미만이면 터미널에서 다음 명령어들을 한번에 한 줄씩 실행하세요:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
\. "$HOME/.nvm/nvm.sh"
nvm install 22

설치 완료 확인

설치가 완료되면 이 명령어로 확인해보세요:

nvm current
npm --version

성공하면 이런 메시지가 나타납니다:

터미널 - 성공적인 경우
사용자명@MacBook-Pro ~ % nvm current v22.17.0 사용자명@MacBook-Pro ~ % npm --version 10.9.2 사용자명@MacBook-Pro ~ %

결과를 선택하세요:

문제 해결

Node.js가 인식되지 않는 경우:

1. 먼저 Node.js 설치 확인:

node --version

2. 설치되지 않았다면 PATH 설정 문제일 수 있습니다:

💡 Apple Silicon Mac (M1/M2/M3/M4):

echo 'export PATH="~/.nvm/versions/node/v22.17.0/bin:$PATH"' >> ~/.zshrc

💡 Intel Mac:

echo 'export PATH="~/.nvm/versions/node/v22.17.0/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

문제가 해결되었다면:

문제 해결 완료!
Node.js가 정상적으로 설치되었습니다

Claude Code 설치

이제 Claude Code를 설치합니다:

npm install -g @anthropic-ai/claude-code

💡 ".. packages are looking for funding" 메시지는 정상입니다. 오픈소스 프로젝트에서 기부를 받는다는 뜻입니다.

설치 완료 확인

설치가 완료되면 이 명령어로 확인해보세요:

claude --version

성공하면 이런 메시지가 나타납니다:

터미널 - 성공적인 경우
사용자명@MacBook-Pro ~ % claude --version claude-code/1.0.51 사용자명@MacBook-Pro ~ %

결과를 선택하세요:

문제 해결

Claude Code가 인식되지 않는 경우:

1. 먼저 설치 확인:

npm list -g @anthropic-ai/claude-code

2. npm 글로벌 경로 확인:

npm config get prefix

3. PATH에 npm 글로벌 경로 추가:

echo 'export PATH="$(npm config get prefix)/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

4. 그래도 안 되면 재설치:

npm uninstall -g @anthropic-ai/claude-code && npm install -g @anthropic-ai/claude-code

5. 터미널을 완전히 종료하고 다시 열어보세요.

문제가 해결되었다면:

문제 해결 완료!
Claude Code가 정상적으로 설치되었습니다

문제 해결

📁 "Permission denied" 오류가 나타날 때:

권한 문제입니다. 홈 폴더에서 시작하세요:

cd ~
mkdir my-project && cd my-project
claude

💡 홈 폴더는 항상 권한이 있으니 거기서 시작하세요!

🌐 "Cannot connect" 오류가 나타날 때:

네트워크 연결 문제입니다. 아래 항목들을 확인하세요:

  • Wi-Fi 확인 - 인터넷 연결되었나요?
  • 브라우저 테스트 - 구글 접속되나요?
  • 터미널 재시작 - 터미널 닫고 다시 열어보세요

🔄 그래도 안 될 때:

가장 간단한 해결책을 시도해보세요:

  • 터미널 완전히 종료
  • 컴퓨터 재시작
  • 터미널 다시 열고 아래 명령어 실행
cd ~ && claude

문제 해결 완료!

Git for Windows 확인

먼저 Git이 이미 설치되어 있는지 확인합니다:

git --version

버전이 표시되면 다음 단계로 가세요. 오류가 나면 아래 설치 과정을 진행하세요.

Git for Windows 다운로드 및 설치

아래 링크를 클릭하세요:

👉 공식 Git 다운로드 페이지

"Click here to download" 버튼을 클릭하면 자동으로 다운로드가 시작됩니다.

💡 다운로드 완료 후 설치 파일을 실행하고, 모든 화면에서 "Next" 버튼만 클릭하세요. 아무것도 변경하지 마세요!

설치 확인

설치가 완료되면 명령 프롬프트를 다시 열고 확인합니다:

git --version
명령 프롬프트 - 성공적인 경우
C:\Users\사용자명> git --version git version 2.50.1.windows.1

결과를 선택하세요:

문제 해결

명령 프롬프트를 다시 시작했는데도 git을 찾을 수 없는 경우:

  • 1단계: 모든 명령 프롬프트 창을 닫으세요
  • 2단계: 컴퓨터를 재시작하세요
  • 3단계: 명령 프롬프트를 다시 열고 git --version 입력

💡 대부분의 경우 컴퓨터 재시작으로 해결됩니다!

재시작해도 해결되지 않는 경우:

Git을 제거하고 다시 설치하세요:

  • 시작 메뉴 → 설정 → 앱 → Git 찾아서 제거
  • 2단계로 돌아가서 다시 설치
  • 설치 시 모든 기본 옵션 유지 (변경하지 마세요!)

설치 중 오류가 발생하는 경우:

관리자 권한으로 설치 파일을 실행하세요 (마우스 오른쪽 클릭 → "관리자 권한으로 실행")
